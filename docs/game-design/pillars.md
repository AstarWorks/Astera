# Pillars

Astera の 3 本柱 (Siege / UGC / Place) の定義と相互関係。

## 1. Siege Warfare PvP

**何**: 攻撃側 vs 防衛側の非対称大規模 PvP。

**ゴール**: プレイヤーに「Astera ならではの試合体験」を提供する。Hypixel BedWars のように 1 度乗り換えたら他に戻れなくなる中毒性。

**主要要素**:
- コア破壊 (RTM の BreakCore モードを発展)
- 攻撃/防衛の役割固定 (1 試合内では切り替えない)
- 構造物破壊 (制限付き)
- 武器の 3 アーキタイプ (銃 / 剣 / 魔法) でロール分担
- 試合長 15〜30 分 (RTM の 10〜15 分 から拡大)

詳細: [[siege/overview]]

## 2. Community UGC

**何**: ユーザーが武器・スキン・マップ・サウンド・スキルを投稿し、自動レビューを経て世界に追加できる仕組み。

**ゴール**: コンテンツ生成を運営から community へ。"参加できる" こと自体が遊びになる。

**主要要素**:
- YAML だけで武器追加 (Phase 1 で動く)
- Discord/Web 経由で git なし投稿 (Phase 4)
- AI モデレーション + 人間レビュー
- マーケットプレイス (Phase 4)
- クリエイター収益還元 (Phase 5+)

詳細: [[ugc/overview]]

## 3. Metaverse Place

**何**: 試合外でプレイヤーが恒久的に滞在・建築できる常設ワールド。

**ゴール**: 「Siege だけ遊んで帰る」から「Astera に住んでる」に転換する。

**主要要素**:
- プロット (個人/ギルド所有区画)
- SNS 的要素 (アルバム / Discord 連携)
- イベント (季節 / 限定試合 / フェス)
- 経済 (通貨流通 / ガチャ / サブスク)

詳細: [[place/overview]]

## 3 本柱の関係

```
              ┌─────────┐
              │  Place  │  ← プレイヤーが住む
              └────┬────┘
                   │ 入口/参加導線
       ┌───────────┼───────────┐
       ▼           │           ▼
   ┌────────┐  ┌────────┐  ┌────────┐
   │ Siege  │  │ Events │  │ Daily  │  ← 遊ぶ場
   └────────┘  └────────┘  └────────┘
                   ▲
                   │ コンテンツ供給
              ┌────┴────┐
              │  UGC    │  ← community が作る
              └─────────┘
```

- **Place は入口と居場所**: 試合が無い時もログインする理由
- **Siege は主要 gameplay**: 1 セッションの中心
- **UGC は無限の供給**: 武器・マップ・スキンが community から尽きず出る
